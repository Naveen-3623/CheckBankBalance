Project: CheckBankBalance


Overview:


A simple Java application that uses JDBC to connect to Oracle 11g XE database to check bank account balance and perform money transfers with proper exception handling.

Technologies Used:


Java,

JDBC,

Oracle 11g XE,

Eclipse IDE.

Features:


✔ Check Account Balance

✔ Transfer Money

✔ Custom Exception (InsufficientFundsException)

✔ JDBC Database Connectivity

How to Run:

Install Oracle 11g XE and create required tables.

Configure database details in DBUtil.java.

Compile the project.

Run BankMain.java.
